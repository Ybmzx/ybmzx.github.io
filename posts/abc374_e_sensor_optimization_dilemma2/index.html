考虑使用二分找出最大生产能力, 每次判断在只有 $X$ 日元的情况下是否能到达生产能力 $mid$.

数据范围:

  • $1 \le N \le 100$
  • $1 \le A_i,B_i \le 100$
  • $1 \le P_i,Q_i,X \le 10^7$

因为一机器一天最多加工 $100$ 产品, 而一个机器最少也要一元钱, 我们最多拥有 $10^7$ 日元, 所以生产能力最多不超过 $100 \times 10^7 = 10^9$, 即生产能力的区间为 $[0, 10^9]$.

接下来看如何计算出到达 $mid$ 生产能力的情况下需要的钱还最少. 因为生产能力的定义为 $\displaystyle \min^{N}_{i=1} W_i$, 其中 $W_i$ 代表第 $i$ 道工序的产能. 可以发现到达 $mid$ 生产能力需要让每一道工序的产能都 $\ge mid$.

可以发现优先采用性价比高的机器更好, 但是不能全部选用性价比高的机器, 如对于以下样例:

$S_i$ 每天可处理 $5$ 个产品, 每台成本为 $5$ 日元.

$T_i$ 每天可处理 $2$ 个产品, 每台成本为 $3$ 日元.

$mid = 2$

显然 $S_i$ 的性价比高, 但是选择一个 $S_i$ 需要 $5$ 日元, 而选择一个 $T_i$ 却只需要 $3$ 日元. 不过我们可以发现, 在不超出需要产能的情况下, 优先选性价比高的机器更好.

也就是假设满足产能大于 $mid$ 时最少花费的情况是买了 $n$ 个机器 $S$, $m$ 个机器 $T$, 其中 $S$ 的性价比大于 $T$, $S$ 的产量为 $SM$, $T$ 的产量为 $ST$. 此时满足 $SM \times n \le mid, SM \times (n + 1) \ge mid$, 也就是 $ST \times m \le SM$

所以我们遍历用 $0$ 台产量低的机器 $T$ 的情况, $1$ 台 $T$ 的情况, $2$ 台 $T$ 的情况… 并从中找到最小花费即可.

那么我们需要遍历多少种情况呢? 因为在不超出需要产能的情况下, 优先选性价比高的机器更好. 假设性价比高的机器每天能处理的产品为最大值: $100$ (数据范围规定), 且性价比低的机器每天能处理的产品为最小值: $1$. 此时: $$ \begin{align} ST \times m &\le SM \\ 1 \times m &\le 100 \\ m &\le 100 \end{align} $$ 所以需要遍历 $[0, 100]$ 的情况.

#include <iostream>
using namespace std;
#define int long long
const int N = 110;
int n, x, A[N], P[N], B[N], Q[N];
signed main() {
    cin >> n >> x;
    for (int i = 0; i < n; i ++)
        cin >> A[i] >> P[i] >> B[i] >> Q[i];

    auto check = [&](int mid) -> bool {
        // 全部工序花费最小值
        int w = 0;
        for (int i = 0; i < n; i ++) {
            // 当前工序花费最小值
            int curr = 1e18;
            for (int cnt = 0; cnt <= 100; cnt ++) {
                int acnt = cnt;
                // r = 买 acnt 台 A 的情况下还需要多少产能才能保证第 i 工序的产能 >= mid
                int r = max(0ll, mid - acnt * A[i]);

                // bcnt = 至少买 bcnt 台 B = r / B[i] 向上取整
                int bcnt = (r + B[i] - 1) / B[i];

                curr = min(curr, bcnt * Q[i] + acnt * P[i]);

                // 上面的反过来
                bcnt = cnt;
                r = max(0ll, mid - bcnt * B[i]);
                acnt = (r + A[i] - 1) / A[i];

                curr = min(curr, bcnt * Q[i] + acnt * P[i]);
            }
            w += curr;
        }
        // 当前最小花费 w 要小于等于 x 说明产能(mid)可能可以更大
        // 即最大产能 >= mid
        return w <= x;
    };

    int l = 0 - 1, r = 1e9 + 1;
    while (l + 1 != r) {
        int mid = l + r >> 1;
        if (check(mid)) l = mid;
        else r = mid;
    }
    cout << l;
    return 0;
}
